Agenda in cloud shortcodes for Wordpress.

== Description ==

This plugin allows Agenda in cloud users to use shortcodes in their wordpress site to show online booking information and timetbale of courses.

== Frequently Asked Questions ==

= How can I use the plugin? =

All instruction to use the plugin are accesible in Agenda in Cloud organization area under the online menu.
